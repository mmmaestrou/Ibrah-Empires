# Fresh Farm Groceries 🌱

Farm-fresh produce delivered straight to your doorstep! A mobile-friendly e-commerce website for fresh fruits, vegetables, and African leafy greens.

![Fresh Farm Groceries](https://via.placeholder.com/1200x600/2e7d32/ffffff?text=Fresh+Farm+Groceries)

## 📋 Features

- 🛒 **Product Catalog** - Browse fresh produce with prices
- 🔍 **Category Filtering** - Shop by vegetables, fruits, legumes, etc.
- 💰 **Discount Display** - See savings on each product
- 📱 **Mobile Responsive** - Works perfectly on phones
- 🚚 **Delivery Info** - Same-day delivery in Nairobi
- 💚 **Sustainable Focus** - Farm-fresh, no harmful chemicals
